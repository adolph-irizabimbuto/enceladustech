August 27, 2024 / by Adolph Irizabimbuto

Set up FAZ on Proxmox

Following this guides:

Step 1: Download the image from Sorti Support
Step 2: Copy the image into the Proxmox server
scp fortios.qcow2 root@x.x.x.x:/root/
Step 3: Import the disk image to the VM
ssh root@proxmox-ip

qm importdisk xxx fortios.qcow2 local-lvm

Where xxx is the newly created VM ID and local-lvm is the Data Store
